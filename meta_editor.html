Mycroft Skill Readme Generator

This tool generates a README.md for your Mycroft skill repository on GitHub. It will be machine-parsed to fill the Mycroft Skill Marketplace and will explain your skill to users in a standardized, easy to read format.

how it works

Edit the form
Generate and examine previews
Copy the output into a README.md file for your skill's repo
  • Enter a brief, speakable skill name. Ex: "Podcast player"
  • Enter a one-line description of your skill. Ex: "Play and subscribe to podcasts"
  • Free form, multi-paragraph description of the skill. You can use Github markdown.
  • Provide examples of phrases handled by the skill after the wakeword is spoken. The first may be highlighted and should be a prime example.
    Ex: "play the news", "set a 10-minute timer"
  • Credits for people or organizations involved (names, emails, and/or GitHub @handles).
  • If only for specific plaforms, check one or more (blank means any platform).
  • Assign 1 or 2 categories to your skill so it is easy to find in the Skill Marketplace.
  • Go to Font Awesome, pick a free, fitting icon and copy its code here. Ex: <i class="fas fa-robot"></i>
  • Enter a HEX, which has good contrast on a white background. Mycroft recommends using an appropriate color from the Mycroft palette. Ex: #FD9E66.
